录记二哈先生 && 二哈先生的小仙女的幸福生活的点点滴滴

2019年1月31日 认识

熊春霞姐: 介绍二哈和小仙女认识。

2019年2月2日 初见

中午, 二哈开车在县广播局接到了小仙女，送了两支玫瑰。
二哈 && 小仙女 在金科【初见小酒馆】见面, 第一次约饭，一起共享午餐。
二哈 && 小仙女 逛了江边公园。
二哈 去了小仙女家, 吃了晚餐、早餐、中餐。

2019年2月3日

二哈 && 小仙女在上午逛了两江广场，逛了县博物馆。
二哈在小仙女家吃完午饭，归家。

2019年2月4日

除夕，在家吃完团年饭

2019年2月5日

春节，大年初一，二哈、小仙女聊了很久很久的微信，畅谈未来。
小仙女 && 周阿姨邀请二哈去家里做客。

2019年2月6日

二哈准备了腊猪腿、腊香肠、桂林竹筒酒、无糖核桃奶去拜访小仙女家人。
二哈中午和小仙女、周阿姨共进午餐, 周阿姨做了很多很多好吃的，给二哈夹了很多很多菜。
二哈 && 小仙女 下午逛了外滩、放了风筝、逛了四方井公园。
二哈 在 小仙女家留宿。

2019年2月7日

早起。
二哈给小仙女下了面条。
二哈 && 小仙女 驱车去了龙缸国家地质公园，拍了很多很多合照、照片。
周阿姨准备了丰富的晚餐，一起共进晚餐
二哈 在 小仙女家留宿。

2019年2月8日

二哈 && 小仙女回了二哈家，拜访了 二哈的爸爸、大姑姑，小仙女给二哈的爸妈包了800元新年红包。
二哈 && 小仙女一起饿肚子，吃面包，二哈送小仙女 回爷爷奶奶家。

在梨坪，春节假期里，二哈和小仙女作别。

2019年2月9日 ~ 至今

二哈已回到北京工作，小仙女将工作迁回县城。
二哈、小仙女 每天都会幸福的视频、快乐的聊天。
二哈的四婶代二哈参加了小仙女大哥的婚礼。
小仙女喝竹筒酒、微嗨。
