上下文管理器（Context Manager）

上下文管理器是指在一段代码执行之前执行一段代码，用于一些预处理工作；执行之后再执行一段代码，用于一些清理工作。

比如打开文件进行读写，读写完之后需要将文件关闭。又比如在数据库操作中，操作之前需要连接数据库，操作之后需要关闭数据库。
在上下文管理协议中，有两个方法__enter__和__exit__，分别实现上述两个功能。
